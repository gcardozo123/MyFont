Welcome to MyFont Project.

MyFont was my final thesis for the Computer Science bachelor degree at Federal University of Santa Maria - Brazil, it is a font system for Odroid-Show. The system is composed by the following programs:

MyFont.sln

MyFont.sln is a Visual Studio 2013 solution that comes with 2 projects: MyFont and MyFontGL, you can use anyone, the only difference is that MyFontGL offers a better visual representation (in OpenGL) of the generated glyphs. Either MyFont or MyFontGL generates 2 new font files, which are: "FontFamilyName_Size_Bit_a_Bit.txt" and "FontFamilyName_Size_Alternado.txt".

Input: a Monospaced TrueType font file. You can find some under MyFont/resources or MyFontGL/resources.

Output: 2 new font files, one for mode Bit a Bit and other for mode Alternado.

FontFamilyName_Size_Bit_a_bit.txt

This is a font file generated by MyFont program.

  • 1st line: Header with file that generated this font, size, glyph dimensions and mode.
  • 2nd line: Glyph dimensions
  • 3rd line: Glyph length
  • 4th and subsequent lines: each line is an array of bytes that represent one glyph. The bytes are in decimal representation.

FontFamilyName_Size_Alternado.txt

This is a font file generated by MyFont program.

  • 1st line: Header with file that generated this font, size, glyph dimensions and mode.
  • 2nd line: Glyph dimensions
  • 3rd line: Glyph length
  • 4th line: an array of bytes that represent one glyph. The bytes are in decimal representation.
  • 5th and subsequent lines: alternates between length of the glyph and the array that represents the glyph itself.

Odroid_Stuff

Odroid_Sketches: Inside this folder you can find 2 Odroid-Show sketches, one for each font mode (Bi a Bit and Alternado). You need to upload to Odroid-Show the sketch that refers to the font mode you want to use.

Adafruit_ILI9340: This is a modified version of the original Adafruit, it includes methods to print the glyphs that I generated.

SerialCom__

This is a Code::Blocks project used to send glyphs to Odroid-Show. It uses serial communication, which means that you need to set correct serial port being used by Odroid-Show.

Input: a font file generated by MyFont.
